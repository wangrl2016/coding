# coding

Personal coding exercises (个人代码练习)

* 使用`git_sync_deps.py`下载C/C++程序需要的第三方库。

* 每个项目下的`run.py`自动运行示例程序。

* `Dockerfile`展示项目构建过程。

```
sudo docker build -t coding-develop:0.1.0 .
sudo docker run -it coding-develop:0.1.0 /bin/bash
```
